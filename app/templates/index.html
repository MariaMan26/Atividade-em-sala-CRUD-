v1.0  ·  SQLite + Flask

Rotas disponíveis

MétodoRotaAçãoStatus

Exemplos de uso com curl

Listar todos os jogos

curl http://localhost:5000/api/jogos

Buscar jogo por ID

curl http://localhost:5000/api/jogos/1

Inserir novo jogo (POST)

Envie um JSON no body com Content-Type: application/json

curl -X POST http://localhost:5000/api/jogos \
-H "Content-Type: application/json" \
-d "{\"nome\": \"Elden Ring\", \"ano_lancamento\": 2022, \"descricao\": \"RPG de mundo aberto desafiador\", \"desenvolvedora\": \"FromSoftware\", \"genero\": \"RPG\", \"preco\": 249.90, \"quantidade\": 7, \"plataformas\": [\"PS5\", \"PC\", \"Xbox Series X\"]}"

Inserir usando arquivo JSON (forma recomendada)

Em vez de escrever o JSON diretamente no terminal, você pode criar um arquivo .json e enviá-lo. 

# Exemplo de arquivo: novo_jogo.json

{
  "nome": "Elden Ring",
  "ano_lancamento": 2022,
  "descricao": "RPG de mundo aberto desafiador",
  "desenvolvedora": "FromSoftware",
  "genero": "RPG",
  "preco": 249.90,
  "quantidade": 7,
  "plataformas": ["PS5", "PC", "Xbox Series X"]
}

# Envio com curl
curl -X POST http://localhost:5000/api/jogos \
-H "Content-Type: application/json" \
-d @novo_jogo.json

O símbolo @ indica que o conteúdo será lido de um arquivo local.

O arquivo deve estar na mesma pasta onde o terminal está sendo executado, ou utilize o caminho completo do arquivo.

Atualizar jogo (PUT)

Envie apenas os campos que deseja alterar. Os demais serão mantidos.

curl -X PUT http://localhost:5000/api/jogos/1 \
-H "Content-Type: application/json" \
-d "{\"preco\": 199.90, \"quantidade\": 3}"

Atualizar usando arquivo JSON

# arquivo: update.json

{
  "preco": 199.90,
  "quantidade": 3
}

curl -X PUT http://localhost:5000/api/jogos/1 \
-H "Content-Type: application/json" \
-d @update.json

Remover jogo (DELETE)

curl -X DELETE http://localhost:5000/api/jogos/1
